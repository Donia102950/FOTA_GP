Installation Instructions:
0-	Make sure that you have MINGW and it's added to Environment Variables.
1-	install Python 3.8 if not installed.
2-	Add Python to Environment Variables. (ex: C:\Program Files\Python38 )
3-	Run Dependencies.py
4-	Add Python Scripts to Environment Variables. (ex: C:\Program Files\Python38\Scripts)
5-	Run ELFI.exe
